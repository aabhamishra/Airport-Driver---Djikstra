This project will map flights across approximately 10 different airports. The user will type in their location as an airport (MKE, LAX) and the program will return the shortest flight between the two airports Along with this, the program will print out any connecting flights needed to be made. The system also prints the lowest estimated price of a flight between the two loctions. Additionally, the user can get a fun fact about each airports, should they want to! Finally, the system contains a list of the aiports supported by the program for the user's reference.
